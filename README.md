# AI Literacy Quiz

An interactive 15-question quiz to test AI governance knowledge, created by [Mave Social AI](https://mavesocialai.com).

## About

This quiz is part of the **"Keep the AI. Kill the Jargon."** AI literacy series by Sumathi Menon. It covers essential AI concepts across five themes:

- **Foundations** — GPT, LLMs, Tokens
- **How AI Thinks** — RAG, Embeddings, Inference
- **Builder's Toolkit** — APIs, MCP, Synthetic Data
- **Governance & Risk** — EU AI Act, Guardrails, Hallucinations
- **Future & Oversight** — Agentic AI, Model Drift, Human-in-the-Loop
